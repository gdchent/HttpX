第一步,必须有

```
public class Update extends CommonRequest {
    @Override
    public String postfix() {
        return "version/version";
    }
}
```

返回格式数组

```
{
    "code": 1,
    "msg": "",
    "data": []
}
```
方式一
`data` 是数组对象,所以用HttpCommonObjResp<T>`T`就是data数组对象(集合对象)
```
HttpRequestFactory.doPost(new Update(), new ResultCallbackAdaptImpl<HttpCommonObjResp<UpdateResp>>() {

            @Override
            public void doOnResponse(HttpCommonObjResp<UpdateResp> response) {
                Logger.e("t", response);
                集合 = response.getDatas();
            }

            @Override
            public void doOnError(ApiException ex) {
                super.doOnError(ex);
            }
        });
    }
```
